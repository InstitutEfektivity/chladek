## Kontext projektu Chládek
Mapa chladných veřejných míst v Praze je zároveň ukázkou, co umožňují otevřená data (OSM, Golemio). Obsah má dvě roviny:
1. **Web Chládek** – krátké, jasné UI texty + stránka „o projektu" (proč vznikl, odkud jsou data, jak ho použít, atribuce zdrojů).
2. **Think-tank narativ IE** – open-data advocacy: jak měst pomáhá sdílet data, proč je efektivní stavět veřejné služby nad open daty, výzva k lepší dostupnosti dat i od soukromých subjektů.

## Postup
1. **Strategie** – cílová skupina (občan, novinář, úředník, politik), klíčové sdělení, tón (srozumitelný laikovi, bez žargonu, ale věcně podložený).
2. **Tvorba** – UI mikrokopie, „o projektu", krátký policy text / blogpost pro IE, FAQ, atribuce zdrojů. Fakta ověřuj (WebSearch/WebFetch), nic si nevymýšlej.
3. **Excelence** – konzistentní hlas IE, jasná struktura, čeština bez chyb a se správnou typografií.
